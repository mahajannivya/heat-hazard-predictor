# Heat Hazard Predictor 

This is a beginner-friendly Machine Learning project that predicts if it's safe to go outside during summer, based on:

- Day of the Week
- Time of the Day
- Temperature (°C)

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Google Colab

##  Sample Output

Input:
- Day: Tuesday
- Time: Afternoon
- Temperature: 41°C

Output:
-  Unsafe to go out
- Precautions:
  - Carry water
  - Avoid direct sun
  - Wear a cap/sunglasses
